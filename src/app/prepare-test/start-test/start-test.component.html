Paso 2:Prepararse para la prueba

Recomendaciones

Antes de empezar, lea con atención los siguientes recomnedaciones para obtener resultados más precisos:

1.- No utilice el dispositivo si el nivel de la bateria es menor al 50%.
2.- Ajuste correctamente el dispositivo para obtener valores más precisos.
3.- La prueba debe tener una duración mínima de 20 minutos.
4.- Ante cualquier interrupción se recomienda iniciar la prueba de nuevo.
5.- Mantenga un ambiente libre de estrés y distracciones.
