Iniciar prueba


Antes de empezar, lea con atención los siguientes recomnedaciones para obtener un resultado más preciso

1.- No utilice el dispositivo si la bateria se encuentra a menos del 50% de su capacidad.
2.- Ajuste correctamente el dispositivo para obtener valores precisos.
3.- La prueba debe tener una duración mínima de 20 minutos.
4.- Ante cualquier interrupción se recomienda iniciar la prueba de nuevo.
5.- Mantenga un ambiente libre de estrés y distracciones.
